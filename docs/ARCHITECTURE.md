# Neuron Forge — Platform Architecture

## Overview

Neuron Forge is a **multi-agent platform** where each agent is a self-contained feature with its own:
- Modal component (`components/XxxModal.tsx`)
- API route(s) (`app/api/xxx/route.ts`)
- Documentation (`docs/agents/xxx.md`)

Agents are independent by design. They share only two things:
1. **Settings** — credentials stored in `data/settings.json` via `lib/settings.ts`
2. **Forge Tools Registry** — `data/forge-tools.md`, read by the Consulting Agent to recommend relevant tools

---

## Request Flow

```
User interaction (Modal)
        ↓
Next.js API Route
        ↓
Claude Sonnet 4.6 (via Anthropic SDK)
        ↓
Optional: Playwright (screenshots / PDF)
Optional: External API (Instagram Graph, WhatsApp, Vercel)
        ↓
Response → Modal displays result
```

## Shared Infrastructure

### `lib/settings.ts`
Central store for all credentials. Reads from `data/settings.json`, falls back to environment variables.

```ts
getAnthropicKey()         // Required for all agents
getVercelToken()          // Optional — auto-deploy websites
getInstagramToken()       // Instagram Posts agent
getInstagramAccountId()   // Instagram Posts agent
getWhatsAppPhoneNumberId() // WhatsApp agent
getWhatsAppAccessToken()  // WhatsApp agent
getWhatsAppVerifyToken()  // WhatsApp agent
```

When adding a new agent that needs credentials, add getters here and expose them via `app/api/settings/route.ts`.

### `data/forge-tools.md`
The Forge agent registry. The **Consulting Agent reads this file at runtime** to know which tools exist and when to recommend them. No code changes needed — just update the markdown when a new agent is added.

### `outputs/redesigns/`
HTML files generated by the website agents, served via `/api/preview/[id]`.

## Token Strategy

Claude Sonnet 4.6 has a context window limit. Each agent manages this differently:

| Agent | Strategy | Max tokens |
|---|---|---|
| Analyze & Redesign | Single pass | 8 000 |
| Create from Google Maps | Two-pass (Part 1 + Part 2) | 4 000 + 4 000 |
| Instagram Posts | Single pass | 2 000 |
| WhatsApp Agent (replies) | Single pass, last 20 messages history | 400 |
| Consulting — Questions | Single pass | 1 500 |
| Consulting — Plan | Single pass | 3 000 |

**Critical rule:** Never use `<style>` blocks in generated HTML — always use Tailwind CDN. CSS generation exhausts tokens before the HTML body is complete, causing blank pages.
